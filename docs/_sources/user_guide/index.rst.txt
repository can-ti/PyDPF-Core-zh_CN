==========
User guide
==========

PyDPF-Core 是用于访问 DPF 后处理功能的 Python 客户端 API。 ``ansys.dpf.core`` 软件包通过 Python 语言能够高效执行计算、定制和远程后处理。

本节的目标是：

- 描述最常用的 DPF 对象实例，以及它们如何帮助您访问和修改求解器数据。
- 提供处理最常见案例的简单方法。
