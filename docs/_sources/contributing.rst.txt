.. _contributing:

==========
Contribute
==========

关于向 PyAnsys 代码库贡献的总体指导出现在 *PyAnsys 开发者指南* 中的 `Contribute <https://dev.docs.pyansys.com/overview/contributing.html>`_ 中。
在尝试贡献 PyDPF-Core 之前，请确保您已经完全熟悉了该指南。
 
以下贡献信息仅适用于 PyDPF-Core。

Clone the repository
--------------------

在开发模式下，通过运行此代码克隆并安装最新版本的 PyDPF-Core：

.. code::

    git clone https://github.com/ansys/pydpf-core
    cd pydpf-core
    pip install -e .


Post issues
-----------
使用 `PyDPF-Core Issues <https://github.com/ansys/pydpf-core/issues>`_ 页面提交问题、报告错误和申请新功能。

如需联系项目支持团队，请发送电子邮件至 `pyansys.core@ansys.com <pyansys.core@ansys.com>`_ 。

View documentation
------------------
PyDPF-Core 最新稳定版的文档托管在 `PyDPF-Core Documentation <https://dpf.docs.pyansys.com/>`_ 。

在文档标题栏的右上角，提供了一个选项，用于在查看最新稳定版本文档和查看开发版本或以前发布版本的文档之间进行切换。
